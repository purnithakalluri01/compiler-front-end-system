
# 🚀 Compiler Front-End System

A mini compiler frontend built using **Python Flask + JavaScript + Monaco Editor**.

---

## 🔥 Features

- Lexical Analysis
- Syntax Analysis
- Semantic Analysis
- AST Generation
- Symbol Table Construction
- Monaco Editor Integration
- Real-time Compilation UI
- AST Visualization (Tree View)
- PDF Report Export

---

## 🧠 Tech Stack

- Python (Flask)
- JavaScript (Vanilla)
- Monaco Editor
- HTML / CSS
- ReportLab (PDF generation)
